# migrate-v1-to-v2

Migration kit for QuantRocket version 2.

## Clone in QuantRocket

CLI:

```shell
quantrocket codeload clone 'migrate-v1-to-v2'
```

Python:

```python
from quantrocket.codeload import clone
clone("migrate-v1-to-v2")
```
